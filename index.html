📋 Setup Guide

1
Create a Google Sheet — Go to sheets.google.com, create a new spreadsheet named THP Attendance.
2
Open Apps Script — Click Extensions → Apps Script. Delete existing code.
3
Paste Code.gs — Copy the full contents of Code.gs and paste it. Save.
4
Deploy as Web App — Deploy → New Deployment. Type: Web App. Execute as: Me. Access: Anyone. Authorise.
5
Paste URL above — Copy the Web App URL and paste it in the field above. Save & Connect.
6
Email notifications — Add staff email addresses in Staff Management. The system will email clock-in/out confirmations and leave decisions automatically via Gmail.
